✅ Code 1: HTML + CSS + JavaScript Password Generator
Objective: To generate secure and customizable passwords directly in the browser based on user preferences.
Tools: HTML5, CSS3, JavaScript (DOM manipulation, Math.random).
Outcome: A responsive web-based password generator that creates instant passwords using selected character types.

✅ Code 2: Python Console-Based Password Generator
Objective: To generate strong random passwords through a command-line application.
Tools: Python, random module, string module.
Outcome: A terminal-based password generator that produces secure passwords based on user input.
